# Goal Creator MVP

## Canonical Goal

Objective:
Create the first working version of goal-creator as a concise multi-format skill that saves goal files under `.goals/`.

Verify:
- Create SKILL.md, scripts/save_goal.py, README.md, and one example file.
- Run `python scripts/save_goal.py --help` successfully.
- Save one sample goal and confirm the file exists under `.goals/`.

Boundaries:
- Write only inside `C:\goal-creator`.
- Do not modify existing installed skills during repository implementation.

Stop:
- The repository contains the skill, save script, example, and a verified saved goal file.

Pause:
- GitHub publishing credentials or repo ownership decisions are required.
